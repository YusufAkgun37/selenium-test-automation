# Selenium Test Automation Project

This project demonstrates a basic QA automation framework built with Java, Selenium WebDriver, and TestNG.

## Technologies
- Java
- Selenium WebDriver
- TestNG
- Maven

## Project Structure
- BaseTest for WebDriver setup
- Page Object Model ready structure
- Smoke and functional test examples

## Test Scenarios
- Open Google homepage and verify page title

## How to Run
1. Clone the repository
2. Navigate to project directory
3. Run `mvn test`
